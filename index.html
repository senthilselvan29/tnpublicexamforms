✨ App Features

  • One-Time Data Entry: Input staff details just once to instantly auto-populate multiple registers and cover pages.
  • All-in-One Platform: A unified web app designed to seamlessly handle forms for 10th (SSLC), 11th Arrear, and 12th (+2) public exams.
  • Smart Automation: Features automatic remuneration calculations, alongside the instant generation of attendance registers, attendance certificates, and more.

⚠️ Please Note

  • The 'Save Locally' button stores your data only within your current web browser and computer. Please do not clear your browser history. If you need to work on a different computer, click 'Download Full Backup' to save your data as a file, then use 'Upload Backup' to restore it on the new machine.
  • Use your browser's Margin and Scale print settings if a table doesn't fit on one page perfectly.
  • Add employee details one by one by categorise. You can select the specific working dates for each employee later using the Edit option.
